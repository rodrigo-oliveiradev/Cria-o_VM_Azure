Projeto: Criação e Gerenciamento de VM no Azure

Este projeto demonstra o processo completo de criação, inicialização e exclusão de uma Máquina Virtual (VM) no Microsoft Azure, seguindo as orientações oficiais da Microsoft:
https://learn.microsoft.com/pt-br/azure/virtual-machines/windows/quick-create-portal

Além disso, documenta as evidências por meio de imagens capturadas durante cada etapa, que estão disponíveis na pasta /Images deste repositório.
